What is It?

This is a tiny extension of jQuery UI Sortable. By default the sortable works off an unordered list. This is great but it doesn't offer any options to integrate the element into a form.

This extension gives you an option to connect a HTML multiselect element to the sortable element. This allows you to easily submit the values from one or more sortable lists with the rest of a HTML form.

Connected Multiselects Using Multiselect Values

Sort items from one list into another and vice versa, by passing a selector into the connectWith option.

Connect the multiselect elements with the connectMultiSelect option. Anytime an item is added, removed or sorted in the sortable element the same will happen in the multiselect element.

The sortable elements get populated using the options in the select elements which are selected using the value set in the connectMultiSelect option.

data-li-class is used to apply a style to the list items in the sortable element.

      JavaScript:

      
    //Connected Multiselects Using Multiselec Values
    $( "#sortable1" ).sortable({
        connectWith:        ".connectedSortable1",
        connectMultiSelect: "#multiselect1"
    }).disableSelection();

    $( "#sortable2" ).sortable({
        connectWith:        ".connectedSortable1",
        connectMultiSelect: "#multiselect2"
    }).disableSelection();

      HTML:

      
    <div class="ul-container">
        <ul id="sortable1" class="connectedSortable1 ui-sortable"></ul>
        <ul id="sortable2" class="connectedSortable1 ui-sortable"></ul>
    </div>
    <div class="select-container">
        <select id="multiselect1" name="multiselect1" multiple>
            <option value="list-a-item-1" data-li-class="ui-state-default">List A Item 1</option>
            <option value="list-a-item-2" data-li-class="ui-state-default">List A Item 2</option>
            <option value="list-a-item-3" data-li-class="ui-state-default">List A Item 3</option>

      JavaScript:

      
    //Custom Functionality after Transition
    $( "#sortable5" ).sortable({
        connectWith:        ".connectedSortable3",
        connectMultiSelect: "#multiselect5",
        selectreceive:      function (e, ui) {
            alert('I (multiselect5) received an item!');
        },
        selectremove:      function (e, ui) {
            alert('I (multiselect5) lost an item!');
        }
    }).disableSelection();

    $( "#sortable6" ).sortable({
        connectWith:        ".connectedSortable3",
        connectMultiSelect: "#multiselect6"
    }).disableSelection();

      Options

      Option Description Default
      connectMultiSelect jQuery selector used to connect the multiselect element to the sortable element. null

      Click here for a list of the options provided by jQuery UI Sortable.

      Events

      Event Description
      selectreceive Fires after the select element has received an item.
      selectremove Fires after and item is removed from the select element.

      Click here for a list of the events provided by jQuery UI Sortable.

      Methods

      Method Description
      updateSelect If the multiselect exists in the DOM this updates its options to match the li's in the sortable element.

      Click here for a list of the methods provided by jQuery UI Sortable.
